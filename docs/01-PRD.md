# FUTUREM Sourcing 产品需求文档 PRD

## 1. 产品定位
FUTUREM Sourcing 是一套独立于 FUTUREM ERP 的义乌外贸采购跟单管理系统，面向义乌采购公司、外贸采购代理、外贸公司和市场采购团队。

它不是进销存，不做普通库存业务。核心是：

客户来义乌采购 -> 多市场多店面询价 -> 客户确认 -> 对供应商下采购订单 -> 多张采购订单汇总成客户汇总单 -> 收货/QC -> 装柜/拼柜 -> 出运 -> 财务结算。

## 2. 核心业务流程
BT 采购行程 -> RFQ 客户询价单 -> 供应商报价 -> CO 客户订单 -> PO 采购订单 -> SO 汇总单 -> 收货 -> QC -> 装柜 -> 出运 -> 财务。

## 3. 核心规则
1. 商品资料只保存固定信息，不保存采购价、销售价、包装、体积、重量。
2. 采购价放在 PO 明细。
3. 销售价放在 CO 明细。
4. 包装资料放在订单明细，包括装箱数量、长宽高、单箱 CBM、单箱毛重。
5. 客户收款按 SO。
6. 供应商付款按 PO。
7. 所有单据支持复制，复制后可以增减商品、修改数量、价格、包装。
8. 历史价格和历史包装从历史 CO/PO 明细查询，并支持跳转原单。
9. 所有单号、客户、供应商、商品都可以点击跳转。
10. 所有删除采用软删除。

## 4. 单号规则
单号采用：前缀 + 年月日时分秒。

示例：
- BT20260701142356
- RFQ20260701142356
- CO20260701142356
- PO20260701142356
- SO20260701142356
- RCV20260701142356
- QC20260701142356
- CL20260701142356
- SHP20260701142356

如同一秒生成多张单，可追加两位流水号。
